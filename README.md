# Writing-Style-Persuasion
This repository includes all and analysis code and participant data collectred for the paper "It's a Bit of Me: Predicting Persuasion from Writing Style and Persuader-Receiver Similarity". It contains the following files:

* `data.csv` file containing participant data.
* `Data_log.md` file which describes all columns in the `data.csv` file.
* `Analysis.Rmd` file which contains the R code used for data analysis.
